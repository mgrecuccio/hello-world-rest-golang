# hello-world-rest-golang
A simple hello world rest server written in golang for learning goland

# How to run?

#### clone project
1. `cd $GOPATH/src`
2. `git clone https://github.com/prakashpandey/hello-world-rest-golang.git`

#### install and run

1. `go install ./...`
2. `$GOPATH/bin/hello-world-rest-golang`
